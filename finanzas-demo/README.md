# Finanzas Demo

Demo sanitizada de la app Finanzas para probar desde iPhone.

## Abrir

Cuando GitHub Pages esté activo para este repositorio, abrir:

`https://erich-alt.github.io/Ing-pen/finanzas-demo/`

## Importante

- Esta carpeta no contiene datos reales.
- Los datos ingresados en la demo quedan en `localStorage` del navegador.
- Sirve para probar navegación, botones, guardado local, exportar/importar JSON y estructura base.
- La app completa React se integrará después con la app de Ingresos/Pensión.
